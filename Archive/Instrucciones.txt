"Database" esta carpeta tiene todas las colecciones de la base de datos que supongo
que le pusieron "foro_uttn".

Para actualizar la base de datos ejecutaran el "Importar.exe", esto
importara las colecciones de la carpeta "Database" a su base de datos.

En caso de ya hayan hecho modificaciones a las colecciones de la base de datos ejecutar
el "exportar.exe" y subir el commit para que los demas veamos la actualizacion que
hicieron, cuando hagan el commit poner que cambio hicieron muy resumido y avisar al grupo.
